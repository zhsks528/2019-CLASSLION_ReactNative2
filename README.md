## Prerequisites

- npm >= 5.5.0
- node > 9.3.0

## Git Clone & Move Dir

원격 저장소에서 다운 받아 이동하는 방법입니다.

```sh
git clone https://github.com/zhsks528/2019-CLASSLION_ReactNative2.git
cd 2019-CLASSLION_ReactNative2
```

## Install

필요한 패키지를 설치합니다.

```sh
npm install
```

## Usage

다음 명령어로 프로젝트를 실행합니다.

```sh
npm run start
```
